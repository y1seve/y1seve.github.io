Attributes

元素的属性包含了对于应用来说很有用的信息，所以如何设置和获取元素的属性就显得很重要。

.attr() 方法

.attr()方法可以扮演getter的角色，也可以扮演setter的角色。作为setter，.attr()方法可以既接收一组key value作为参数，也可以接收包含一组或多组key value的object对象。
