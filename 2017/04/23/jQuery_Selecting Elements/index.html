Selecting Elements

选择元素和操作元素是jQuery中最基本的操作。jQuery支持大多数CSS3选择器，和一些非标准的选择器。jQuery支持的完整的选择器可参考官方文档。

判断是否选择到了元素的方式是判断.length属性，如果没有选择到元素，该属性就为0，则会被转换为false。

Saving Selections

jQuery并不会为你缓存元素。如果你做了一次选择操作，下次获取该元素时，你需要再次做同样的选择操作；你应该将选择到的元素保存到一个变量中而不是重复地做选择操作。

一旦选择的元素存储到变量中，则你可以在该变量上调用jQuery方法，就像在原始的选择上调用jQuery方法一样。

选择只会获取到 在做出选择时处于页面上的元素。如果有些元素在这之后添加到页面中，你需要重复再做选择，或者将这些要添加到页面中的元素添加到存储到变量的选择中去。那些存储到变量中的选择是不会在DOM更新的时候神奇地更新的。
